本文档为树模型独立用例树，内容由原总文档拆分保留。
适用于树模型侧的测试设计、覆盖盘点、需求评审与后续补用例排期。

当前状态说明：

- 当前文档范围：Tree_Model
- 当前累计可执行用例：397
- 当前已覆盖：实例管理、登录、首页、测点管理、查询、SQL 操作、可视化、视图、数据同步、权限管理、审计日志、数据库配置
- 当前未覆盖：AI 分析

## 1. 实例管理

### 1.1 单机实例新建

#### 1.1.1 完成单机实例新建流程

- 前置条件
  - 已进入登录页
  - 点击【编辑】进入实例管理
- 操作步骤
  - 点击新增实例
  - 保持连接类型为“单机”
  - 输入实例名称、主机、端口、用户名、密码、Prometheus、默认模型
  - 点击【测试】
  - 点击【保存】
- 预期结果
  - 浮层提示连接成功
  - 实例列表新增对应实例

#### 1.1.2 空表单直接保存触发校验

- 前置条件
  - 已打开新建实例表单
- 操作步骤
  - 不输入任何内容点击【保存】
- 预期结果
  - 实例名称、Host、端口、用户名等必填项出现红字校验

### 1.2 表单交互

#### 1.2.1 重置表单后恢复默认值

- 前置条件
  - 已打开新建实例表单并输入部分内容
- 操作步骤
  - 点击【重置】
- 预期结果
  - 连接类型恢复“单机”
  - 默认模型恢复“树模型”
  - 文本输入项恢复为空

#### 1.2.2 未保存切换实例时触发确认弹窗

- 前置条件
  - 当前表单存在未保存修改
- 操作步骤
  - 切换到其他实例
  - 分别选择“继续编辑”和“放弃修改”
- 预期结果
  - 继续编辑时草稿保留
  - 放弃修改时切换到目标实例

#### 1.2.3 刷新实例列表后仍显示最新保存数据

- 前置条件
  - 已保存至少 1 个实例
- 操作步骤
  - 点击刷新按钮
  - 重新查看目标实例
- 预期结果
  - 列表显示最新实例数据

### 1.3 编辑与删除

#### 1.3.1 编辑已保存实例并持久化

  - 存在可编辑实例

- 前置条件

- 操作步骤

- 预期结果

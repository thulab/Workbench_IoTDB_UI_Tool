## 4. 当前未覆盖清单

### 4.1 优先级 P1

- 编辑视图时结果测点为禁用态的只读校验细节，例如禁止复制后绕过修改
- 视图列表多选后跨分页批量删除行为
- 导出前筛选条件叠加场景，例如“结果测点筛选 + 导出”与“视图描述筛选 + 导出”
- 刷新按钮在无读数据权限、无读写元数据权限下的禁用与 tooltip 差异

### 4.2 优先级 P2

- 编辑弹窗中更多表达式组合，例如减法、函数嵌套、常量表达式
- 批量导入视图的更多异常场景，例如空文件、超大文件、重复点击下一步、XLSX 异常模板
- 分页场景下切换页大小 `10 / 20 / 50 / 100` 的联动验证
- 行内【查看数据】跳转后再回到视图页时筛选条件保持情况

### 4.3 优先级 P3

- 多浏览器项目回归，例如 WebKit / Firefox
- 报告入口按视图页模块单独固化 `headed:report` 的周期性基线归档
- 表模型视图页面骨架与同类用例迁移
